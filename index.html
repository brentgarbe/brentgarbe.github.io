How Latitude Affects the Weather: 500+ cities around the world


This project is to showcase an analysis that we performed on how latitude affects affects the weather across 500+ cities around the world. Previously, using API data from OpenWeather, over 500+ locations were generated from around the world at various latitudes. Weather data was then pulled for each of these locations. The data that was analysed is max temperature, cloudiness, humidity, and wind speed. There are web pages dedicated for each of these analysis as well as a full comparison sheet where all results are compared side by side. Enjoy browsing through my webpages!
